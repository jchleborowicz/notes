[cols="1,10"]
|==================================
|\d         | digit
|\D         | non-digit
|.          | any character
|\w         | alpha
|\W         | non-alpha
|{m}        | m repetitions
|{m,n}      | m to n repetitions
|*          | zero or more
|+          | one or more
|?          | optional character
|\s         | any whitespace
|\S         | any non-whitespace
|^...$      | starts and ends
|(..)       | capture group
|(a(bc))    | capture sub-group
|(.*)       | capture all
|(abc\|def) | matches abc or def
|\b         | boundary between word and non-word character
|==================================
